# AI Second Brain — Telegram Bot

Telegram-бот для хранения заметок, семантического поиска, умных напоминаний
и еженедельных сводок.

## Возможности

- 📝 Заметки из текста, голоса и фото (с распознаванием изображений)
- 🔍 Семантический поиск по заметкам и ответы ИИ с учётом контекста
- ⏰ Умные напоминания: бот сам находит даты/события в заметках и напоминает за сутки
- 🗓 Еженедельная сводка (авто по расписанию + по запросу `/summary`)
- 🌍 Персональный часовой пояс (спрашивается при первом запуске)

## Стек

- **Bot**: aiogram 3.x
- **Обработка текста**: Claude через [routerai.ru](https://routerai.ru) (OpenAI-совместимый API)
- **Эмбеддинги / голос (Whisper) / анализ изображений**: OpenAI
- **Vector DB**: Qdrant
- **DB**: PostgreSQL + asyncpg
- **Планировщик**: фоновый asyncio-цикл (напоминания + сводки)
- **Контейнеризация**: Docker Compose

## Провайдеры ИИ

Бот использует двух провайдеров (см. `.env.example`):

- **RouterAI** (`ROUTERAI_API_KEY`) — Claude (`anthropic/claude-opus-4.8-fast`)
  для всей обработки текста: резюме и теги, ответы, сводки, извлечение напоминаний.
- **OpenAI** (`OPENAI_API_KEY`) — эмбеддинги для поиска, Whisper для голоса,
  vision-модель для анализа изображений.

## Быстрый старт

```bash
# 1. Скопировать конфиг
cp .env.example .env
# Заполнить .env своими ключами

# 2. Запустить всё
docker compose up -d

# 3. Применить миграции
docker compose exec bot python -m scripts.init_db
```

## Использование

- Отправь текст / голосовое / фото → бот сохранит как заметку с автотегами
- Задай вопрос → бот найдёт релевантные заметки и ответит
- Запиши мысль с датой («в среду в 19:00 театр») → бот напомнит за сутки
- `/reminders` — список ближайших напоминаний и удаление ненужных
- `/summary` — сводка за неделю в любой момент
- `/settings` — часовой пояс, день и время авто-сводки
- `/timezone` — сменить часовой пояс

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие, при первом запуске — выбор часового пояса |
| `/help` | Справка с примерами |
| `/reminders` | Ближайшие напоминания |
| `/summary` | Сводка за неделю |
| `/settings` | Настройки: пояс, расписание сводки |
| `/timezone` | Сменить часовой пояс |

## Ограничение доступа

В `.env` указывается `ALLOWED_USER_IDS` — список Telegram ID через запятую.
Бот отвечает только пользователям из белого списка.
